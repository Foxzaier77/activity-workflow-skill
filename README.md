# 活动工作流 Skill

这是一个 Codex Skill，用于 H5 活动从策划到原型、网页 demo、批注、埋点验证和复盘的完整工作流。

## 包含内容

```text
h5-activity-workflow/
  SKILL.md
  agents/
    openai.yaml
```

## 使用方式

把 `h5-activity-workflow` 文件夹复制到你的 Codex skills 目录，例如：

```text
C:\Users\Administrator\.codex\skills\h5-activity-workflow
```

之后在 Codex 中可以说：

```text
使用活动工作流，帮我梳理这个 H5 活动方案。
```

## 能力范围

- 活动启动判断：快速、标准、完整模式
- 一页活动 Brief
- 竞品活动分析
- 活动策划方案
- 产品方案拷打
- 埋点与统计验证
- 活动文案工作流
- Figma 原型流程
- 本地网页 demo
- 网页批注反馈
- Deck / HTML / PDF / PPTX 演示包
- 活动复盘报告

## 维护方式

如果后续使用中发现流程不顺，可以继续迭代 `h5-activity-workflow/SKILL.md`。
